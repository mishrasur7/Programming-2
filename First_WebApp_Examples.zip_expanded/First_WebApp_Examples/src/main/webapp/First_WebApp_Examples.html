FIRST WEB APP EXAMPLES

Sending HTTP GET and HTTP POST requests

  1. When you click on the link below, the browser sends an HTTP GET request for the Web resource. Then, the servlet container invokes the servlet's doGet method. The doGet method creates the response text to be sent to the browser.

    http://localhost:8080/First_WebApp_Examples/firstService

  2. When you click on the link below, the browser sends an HTTP GET request for the Web resource. Then, the servlet container invokes the servlet's doGet method. This time, the request includes two parameters in the URL. You can use the HttpRequest object's getParameter method to obtain the parameter values. Either the method returns the parameter value as a String, or null if the parameter is not part of the request. The doGet method creates the response text to be sent to the browser.

    http://localhost:8080/First_WebApp_Examples/secondServlet?firstname=Joe&title=Mr

  3. When you click on the button below, the browser sends an HTTP POST request for the Web resource http://localhost:8080/First_WebApp_Examples/thirdService. Then, the servlet container invokes the servlet's doPost method. Form parameters are passed to the servlet as name/value pairs in the HTTP POST request. The doPost method creates the response text to be sent to the browser.
